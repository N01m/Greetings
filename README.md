# Greetings

Automatically sends a configurable greeting when you join a party or raid group.

## Features

- Separate messages for party and raid
- Configurable delay (0-30 seconds)
- 10-second cooldown to prevent spam
- Draggable minimap button

## Usage

| Command | Action |
|---|---|
| `/greet` or `/greetings` | Open config panel |
| `/greet on` / `/greet off` | Toggle on/off |
| `/greet test` | Send greeting now (must be in a group) |
| `/greet <text>` | Set message for both party and raid |

Right-click the minimap button to toggle. Left-click to open settings.

## Requirements

- World of Warcraft: The War Within (Interface 120001+)

## Installation

Drop the `Greetings` folder into `Interface/AddOns/` and reload.
