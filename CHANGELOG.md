# Changelog

## [1.0.3]

### Fixed

- Custom messages are now saved when clicking away from the text field, not just on Enter

### Changed

- Removed the Reset to Default button from the settings panel

## [1.0.2]

### Fixed

- Greetings now fire correctly when joining a new group while already grouped

### Changed

- ElvUI users will now see the config panel styled to match their UI

## [1.0.1]

### Fixed

- LFG/Group Finder dungeon groups now correctly wait until you are loaded into the instance before sending the greeting
- Greeting no longer fires when you create a group by yourself with no other members

### Changed

- Default greeting message changed from `o/` to `hi`
- Minimap button icon changed
- Settings panel now shows the current version number in the bottom-right corner

## [1.0.0]

- Initial release
